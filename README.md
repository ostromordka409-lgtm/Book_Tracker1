Описание программы: Графическое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Данные сохраняются в файл books.json. Доступна фильтрация по жанру и количеству страниц. Встроена валидация ввода.

Требования:

Python 3.8+
Библиотека tkinter (входит в стандартную поставку Python)
Инструкция по запуску:

Установите Python (если не установлен).
Скачайте или клонируйте этот репозиторий.
Откройте терминал в папке проекта.
Запустите приложение: python book_tracker.py
Для работы с Git:

Инициализируй репозиторий: git init
Добавь файлы: git add .
Сделай коммит: git commit -m "Initial commit"
Залей на GitHub
Примеры использования:

Добавление книги: введите данные и нажмите «Добавить книгу».
Фильтрация по жанру: выберите жанр в выпадающем списке.
Фильтрация по страницам: введите число и примените фильтр.
